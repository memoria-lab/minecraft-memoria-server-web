# マイクラめもりあ鯖ウェブ

## 環境設定

```
REACT_APP_API_KEY=
REACT_APP_API_BASE_URL=
REACT_APP_GA_ID=
```

## スクリプト一覧

### `npm start`

開発者モードで開発用サーバを起動します。

[http://localhost:3000](http://localhost:3000)へアクセス

### `npm run build-dev`

開発者モードでビルドします。

### `npm run build-prd`

本番モードでビルドします。

### `npm run server-dev`

開発者モードでビルド後、サーバーを起動します。

### `npm run server-prd`

本番モードでビルド後、サーバーを起動します。

### `npm run mock`

REST API のモックを起動します。
